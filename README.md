# CustomCPU
CustomCPU is a project that created custom 16bit CPU in digital logic sim, then added working assembly compiler and in the near future will create working high level code compilator and emulator for faster program run. 
The main purpose of it is to have fun creating something from scratch.

## Digital Logic Sim

## Assembly
You can find whole documentaion on assembly in CPU_documentation file in repo.
Currently assembly contains 32 mnemonics or commands and some of them are empty. It contains most of the useful commands to recreate pretty much anything.
